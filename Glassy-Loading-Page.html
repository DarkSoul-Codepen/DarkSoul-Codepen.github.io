Procedure

Step 1: The HTML Structure

Start by defining the basic structure of your webpage using the <!DOCTYPE html> declaration. This ensures the document adheres to HTML5 standards. Then, use the <html> tag to define the root of the document and specify the language attribute as lang="en" for English.

Step 2: Adding the Head Section

Inside the <head> section, include essential metadata:

  • Charset: Use <meta charset="UTF-8"> to set the character encoding to UTF-8.
  • Viewport: Add <meta name="viewport" content="width=device-width, initial-scale=1.0"> to ensure responsive design on different devices.
  • Title: Set the title of your webpage with the <title> tag, e.g., "Glassy Loading Page."
  • Fonts: Import Google Fonts using @import statements in the CSS. Multiple fonts like "Belanosima," "Rubik Vinyl," and "Kaushan Script" are added for styling flexibility.

Step 3: Crafting the CSS Styles

Next, move to the <style> section to design your page:

  • Body Styling: Set the body to cover the entire viewport with a background image, using background-image, background-position, and background-size.
  • Container Class: Define a .darksoul-container to centralize elements with flexible width and height.
  • Glassy Effects: Add backdrop blur effects for a futuristic look using backdrop-filter and background-color with transparency.
  • Animations: Use @keyframes to create rolling effects for elements, like a glowing square moving along a path. Each animation step transforms the translate and rotate properties.

Step 4: The Animated Content

Inside the <body>, create a div with the class .darksoul-container that houses two main components:

  • The Line: A div with the class .line, styled with glassy effects, contains a heading (<h1>) for the loading message.
  • The Square: A glowing, animated square defined by the .square class, showcasing the rolling animation.

Step 5: Defining Keyframes for the Animation

To create a smooth, looping animation for the square, we use the @keyframes rule. The animation involves translating and rotating the square to give it a rolling effect along a predefined path.

Animation Breakdown:

  • Translate: Moves the square along the X and Y axes to simulate motion.
  • Rotate: Simultaneously rotates the square around its center to enhance the rolling effect.
  • Timing: Each stage of the animation is defined at specific percentages (e.g., 25%, 50%) to control its movement over time.

How It Works:

  • 0%: The square starts at its original position with no rotation.
  • 25%: The square moves 100px to the right and rotates 90 degrees clockwise.
  • 50%: The square moves down 100px and completes a half rotation (180 degrees).
  • 75%: The square moves back to the left and rotates 270 degrees.
  • 100%: The square returns to its starting position and completes a full 360-degree rotation.

Step 6: Final Touches

Finally, combine all these elements to deliver a responsive, visually captivating, and animated loading page.

And that's it! You've built a sleek Glassy Loading Page with mesmerizing animations. Have fun customizing it further!
